# 会战开始结束提醒插件 v 1.1

## 简介

在会战开始前一周会自动提醒会战即将开始，然后会在会战开始前四天艾特会战提醒会长检查成员。

同时支持直接询问会战开始时间和结束时间

## 使用方法

- 将本文件夹放入hoshino/modules文件夹内

- 然后在__bot__.py中加入clanbattle_remind

## 指令列表

|    指令    |    参数    |    效果    |
| :--------: | :----------: | :--------:|
|     设置会长    |    @会长或会长QQ    |    为公会群绑定会长    |
|    公会提醒设置    |    提醒/不提醒会长    |    开启/关闭@会长功能（默认关闭）    |
|    会战还有多久    |        |    查询会战开始/结束天数    |
